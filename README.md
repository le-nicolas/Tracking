# Tracking

Tracking is a lightweight browser app for logging metrics over time.

## What it does

- Create track entries with date, category, label, value, unit, and notes
- Filter and search your timeline instantly
- View quick stats:
  - Total entries
  - Total value
  - Most active category
  - Current streak in days
- Export your records as CSV
- Keep data locally in your browser via `localStorage`

## Run locally

1. Clone the repository.
2. Open `index.html` in your browser.

No build step and no dependencies.

## Files

- `index.html`: UI structure
- `styles.css`: visual design and responsive layout
- `app.js`: app state, storage, rendering, and interactions
